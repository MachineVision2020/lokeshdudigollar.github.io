Scrapping data from Magicbricks

Now-a-days we tend to end up in places far away from home thereby going for rented rooms,PGs etc. With digital world it has become a lot easier to search for rooms using online servies like magicbricks, propertydeal.com etc. But that doesn't make work easier because we often encounter huge number of results when we search for a city or area so going through every result and page is cumbersome.

After encountering such problem myself I built this python script which exstracts the data from Magicbricks and puts that in a nice CSV file with all relavant information. I just need to execute the script online. It was conceptualized with the help of beautifulSoup and Pandas. I do have plan to build a full functional android app where user can just enter the locality and it will give him data in CSV file.

Raspberry Pi based Motion detection

This is a robot powered by RaspberryPi coupled with computer vision.The robot was attached with a camera and whenever any person/moving object was entering within camera's range the robot would detect it and note down the start time when object entered and the end time when object disappeared. This time data was then plotted on website using Bokeh. OpenCV was used for vision.Pandas was used to create CSV file consisting all data
